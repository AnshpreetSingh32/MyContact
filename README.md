# MyContact 📒

**MyContact** is a console-based contact manager application built in **C++**.  
It provides CRUD operations (Create, Read, Update, Delete) and uses file handling to securely store and retrieve contact information.

---

## ✨ Features
- ➕ Add new contacts with name, phone, and email  
- ✏️ Edit existing contact details  
- ❌ Delete contacts from the list  
- 🔍 Search contacts by name or number  
- 💾 File handling ensures data persistence  

---

## 🛠 Tech Stack
- **Language**: C++  
- **Concepts**: OOPs, File Handling
